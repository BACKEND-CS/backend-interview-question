      1. 가상 회선 방식의 연결 지향형 프로토콜(3/4-way handshake)

      5. 속도 느림
     
    - UDP
      1. 데이터그램 방식의 비연결형 서비스
      2. 전송 순서 미보장
      3. 데이터 수신 여부 미확인
      4. 신뢰성이 낮음
      5. TCP보다 속도 빠름
